# Cloud Computing Fundamentals

---

## What you’ll learn

* Define cloud computing and describe its key characteristics 

* Explain how local servers and the internet enable data sharing

* Identify problems that cloud computing helps solve

* Understand different cloud service models and their characteristics

* Compare proprietary and open-source cloud solutions

* Recognize services offered by major cloud providers

* Explain the business value of cloud computing

* Describe on-premises hosting and cloud migration strategies

* Understand public, private, and hybrid cloud deployment models

* Explain virtual resources, containers, and microservices in the cloud

* Understand APIs and virtualization best practices

* Create, install, and test applications using Docker containers

* Understand the process of developing software in the cloud

* Explain the benefits of cloud orchestration

* Build and deploy Docker containers to the cloud

* Identify the importance of cloud security

* Recognize common cloud security vulnerabilities

* Understand identity and access management (IAM) features

* Explain database as a service (DBaaS) and its benefits

* Understand data governance and compliance in cloud databases

* Create and review security settings in an IBM Cloudant database

* Understand the cloud computing job market

* Identify roles, responsibilities, and required skill sets

* Explore learning resources and career opportunities in cloud computing
