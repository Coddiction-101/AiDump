Cloud Computing is a delivery of the computing power including

- Server
- Storage
- Database

(optional)

- Networking
- Intelligence
- Analytics

Over the internet (Cloud). Offers faster innovation, flexible resources and economies of scale. 

*⤴️ Textbook definition*

Imagine, You need electricity, You don’t build own power plant, You pay a utility company for exactly for what you use, when you need. 

Instead of owning a physical server, data center, and an entire infrastructure needed to run a software - You rented it from providers who have already build a globally available distributed Infrastructure, and you can access it over the internet.

Cloud Computing works on the exactly same principal.

Before Cloud If a company want to launch a website, they need to hire a team, purchasing physical server, data center, manage backup, managing cooling system, handle failures, provision storage and scale manually. This required enormous capital investment and months of preparation. With a cloud, you can spin up globally available server exactly in 60sec. and pay only for what you actually use.

This sift represents the significant changes in the history of the technology. It democratized or adjust the access to computing power - making resources available to everyone which was previously available only for the large corporations now available for the startups, students, and indivisual.

Cloud is not just a technology, It is a economic model of computing. It is a infra-as-a-service that delivered with the reliability of the utility.
